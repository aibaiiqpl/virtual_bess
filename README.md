# Virtual BESS

虚拟储能系统模拟器，通过 Modbus TCP 对外暴露 PCS/BMS 控制和状态寄存器，用于开发调试。

## 依赖

- [go-common](https://cnb.cool/aiwatt/ems/go-common)（mbserver、zaplog）

## 配置

编辑 `config.yaml`：

```yaml
bess:
  rated_capacity_kwh: 100    # 额定容量 kWh
  rated_power_kw: 50         # 额定功率 kW
  initial_soc: 50.0          # 初始 SOC %
  soh: 100.0                 # SOH %（固定）
  battery_voltage: 512       # 电池总电压 V（固定）
  grid_voltage: 220          # 三相电压 V（±5% 波动）

modbus:
  address: ":502"

log:
  file: "logs/virtual_bess.log"
  level: "info"
  console: true
```

## 构建和运行

```bash
go build -o virtual_bess .
./virtual_bess -config config.yaml
```

## 操作流程

1. BMS 合闸：写 40001 = 1
2. PCS 开机：写 30003 = 1
3. 下发功率：写 30010 = 500（充电 50kW）或 -500（放电 50kW）
4. 读取状态：读 30061（实际功率）、40105（SOC）等

> 未合闸就启动 PCS 会触发直流侧欠压故障（30180 = 1）。
> PCS 在就地模式（30006 = 0）时功率指令不生效。

## Modbus 点位表

### PCS 控制（可写）

| 地址  | 名称           | 类型 | 备注                          |
|-------|----------------|------|-------------------------------|
| 30000 | 并/离网设置     | U16  | 0-并网 1-离网                 |
| 30001 | 运行模式设置   | U16  | 默认 2（交恒功率）            |
| 30002 | 故障复位       | U16  | 1-复位                       |
| 30003 | 设备开机       | U16  | 1-开机                       |
| 30004 | 设备关机       | U16  | 1-关机                       |
| 30005 | 远程急停       | U16  | 1-急停                       |
| 30006 | 远程/就地设置   | U16  | 0-就地 1-远程，默认 1         |
| 30010 | 充放电功率指令 | S16  | 0.1kW，正充负放              |
